# Labguru (labguru)

Labguru is an electronic laboratory notebook (ELN) and biotech data management platform that provides a unified REST and GraphQL API for managing experiments, protocols, milestones, inventory, equipment, and research project data. The platform serves biotech, pharma, CRO, food tech, and academic organizations with cloud-based lab operations, LIMS, workflow automation, and data analytics capabilities.

## APIs

### Labguru REST API

RESTful API providing create, read, update, and delete access to nearly all data in Labguru including experiments, projects, protocols, milestones, inventory, equipment, measurements, samples, and research data. Supports JSON responses, pagination, filtering, and both v1 and v2 endpoint versions.

- **Human URL:** [https://www.labguru.com/developer-tools](https://www.labguru.com/developer-tools)
- **Base URL:** `https://my.labguru.com/api/v1`
